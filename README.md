# SETUP
## LINUX
setup c-lang, llvm: 
- `sudo apt-get install lld clang`
- `sudo apt install software-properties-common`
- bash -c "$(wget -O - https://apt.llvm.org/llvm.sh)"

setup ssl:
- `sudo apt-get install libsasl2-dev`

## MACOS
setup zld: 
- `brew install michaeleisel/zld/zld`

setup ssl:
- `brew install openssl`
